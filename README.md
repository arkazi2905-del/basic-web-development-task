# 🌐 Personal Portfolio Website

## 📌 Description
This is a simple personal portfolio website built using HTML, CSS, and JavaScript. It showcases basic front-end development skills with a clean and responsive design.

The website includes sections such as Home, About, Skills, and Contact, along with basic interactive features.

---

## 🛠️ Technologies Used
- HTML5 – for webpage structure  
- CSS3 – for styling and layout design  
- JavaScript – for basic interactivity  

---

## ✨ Features
- Responsive navigation bar  
- Hero section with profile introduction  
- About section describing background and interests  
- Skills section highlighting technical abilities  
- Contact form layout  
- Button interaction using JavaScript (alert message)  

---

## 🎨 Design Highlights
- Clean and modern UI  
- Smooth layout structure  
- Soft color theme with hover effects  
- Easy-to-read typography  

---

## 🚀 Purpose
This project was created to practice and demonstrate foundational web development skills and understand how HTML, CSS, and JavaScript work together to build a functional webpage.
